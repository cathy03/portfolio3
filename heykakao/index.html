음성인식

01. 단답형 시나리오

에이전트 웨이크업시 버튼에서 자연스럽게 이어지는 인터랙션으로 브랜드 사용과 연결된 사용자 경험을 제공. 음성 인식중 화면은 어두운 톤을 활용하여 확실하게 구분되어 사용자가 인지 할수 있도록 하였다.

02. 슬롯필링 시나리오

해당 과업을 완료하기 위해 에이전트내에서 추가 정보를 묻고 답변하는 대화의 과정을 표현하기 위해 에이전트가 활성화시 시각 요소를 통일감있게 활용하였다.

음악인식

음성 인식과는 구별되는 인터랙션으로 음악을 인식 중인 상황을 파악 할수 있는 형태로 디자인 하였으며, 음성 인식을 통해 음악인식으로 전환되는 시나리오까지 고려하여 작업하였다.

번역 기능

TYPE A

음성 입력이 아닌 타이핑 기능이 접목된 번역 기능을 어색하지 않도록 엮어내는 것이 미션이었다. 다른 번역 서비스의 익숙한 사용자 경험을 그대로 녹여내는 방향으로 작업하였다.

TYPE B

번역으로 진입하는 애니메이션을 음성, 음악 인식과 동일하게 구성하여 서비스의 일관성을 유지하면서, 타이핑보다 음성을 통한 번역 기능을 강조하는 방향으로 작업하였다.

받아쓰기

TYPE A

대화 내용을 텍스트로 변환하고 화자를 분석할 수 있는 받아쓰기 기능 디자인 및 인터랙션을 작업하였다. 진입하는 애니메이션을 음성, 음악 인식과 동일하게 구성하여 서비스의 일관성을 유지하도록 하였다.

TYPE B

음성 입력 방식이 주된 헤이카카오의 서비스에 맞춰서 녹음과 목록을 나눠서 구성하여 서비스 일관성을 유지하는 방향으로 작업하였다.
